Batch Binary Normalisation

This is used to normalise binary files within a directory. This is also referred to as XML wrapping. For more information on XML wrapping see How to XML wrap a data object  in Basic Plugin.

This process converts each data file in a directory into base64 and places an XML wrapper containing metadata around it. The procedure assumes that you have read the Batch Normalisation Introduction, and are familiar with the layout of the Batch Processing Setup window.

This procedure assumes that you already have the Batch Processing Setup window open. If not, open it now:

  1. open Xena.
  2. from the File menu, select Batch Process.

Before you batch normalise records, you should specify the location of the files you want to normalise and where you want to store the normalised versions. See the Batch Normalisation Introduction. If you have not selected both source and destination directories, do so now.

Selecting the directory

You should first select the directory or directories you wish to XML wrap:

  1. Select the directory you wish to normalise in the left window and click on the »  to move the directory into the right window under the groups folder.
    Repeat this step for each directory that you wish to normalise.
  2. Use the «  or clear buttons to remove a directory from the list of those to be normalised.
    The directories you wish to XML wrap will now appear as groups in the output directory list.

Configuring the normaliser

When XML wrapping the files in a directory, you will need to tell Xena it is XML wrapping all the files in the directory rather than relying on the Xena guesser to choose what to do. To do this, use the Configure Batch Group window:

  1. In the output directory select a group. You may only select one at a time.
  2. Click on the Configure button.
    The Configure Batch Group window will open.
  3. Under File Type Selection, click on Select.
  4. Select the Binary file type from the drop down list.
  5. Xena will automatically select the Binary Normaliser.
  6. Click OK.

Repeat for each group selected.

Running the batch XML wrapping process

Now that the batch process has been configured, you can run the batch process itself. To do this click on the Start button.

To halt the batch process, click on the Stop button. To continue with a halted batch process, click on the Continue button. When you have finished the batch process, click on Close.

As Xena normalises files, the results will appear in the bottom window. When the batch normalisation is complete, the Stop button will grey out and a completion notice - the word DONE - will be appended to the results list in the bottom window. The results window will show OK for each file properly normalised, and ER if there is an error. Any error messages will be appended to the line for that document.
