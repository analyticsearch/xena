Batch normalising a directory

This procedure is used to normalise files in a particular directory and all its sub-directories.

When batch normalising a directory, Xena will guess the most appropriate file format and normalise the records using the relevant normaliser. For more information on how Xena guesses file types see How the Xena application works to preserve digital records.

When Xena does not have a normaliser that is specific to the particular file type within the directory, it will normalise it as a binary file.

It is also possible to configure the batch processor to use a particular normaliser for a batch processing task. This is usually done when normalising binary files, also known as XML wrapping. For instructions on how to select the normaliser for a directory when not XML wrapping see Configuring the Normaliser.

Some of the Xena plugins require you to install extra software, such as OpenOffice.org for Office files. Where extra software is required, it is listed in the help for the plugin. This procedure assumes that you have read the Batch Normalisation Introduction, and are familiar with the layout of the Batch Processing Setup window.

This procedure assumes that you already have the Batch Processing Setup window open. If not, open it now:

  1. open Xena.
  2. from the File menu, select Batch Process.

Before you batch normalise records, you should specify the source directory of the files you want to normalise and the destination directory to store the normalised versions. See the Batch Normalisation Introduction. If you have not selected both source and destination directories, do so now.

Normalising a directory

To conduct batch normalisation of an entire directory use the following procedure:

  1. Select the directory you wish to normalise in the left window, and click on »  to move the directory into the right window under the groups folder.
    Repeat for each directory you wish to normalise.
  2. Use the «  or clear buttons to remove a directory from the list to be normalised.
  3. Click on the Start button to commence batch normalisation.

To halt the batch process then click on the Stop button. To continue with a halted batch process, click on the Continue button. When you have finished the batch process, click on Close.

As Xena normalises files, the results will appear in the bottom window. When the batch normalisation is complete, the Stop button will grey out and a completion notice - the word DONE - will be appended to the results list in the bottom window. The results window will show OK for each file that is properly normalised, and ER if there is an error. Any error messages will be appended to the line for that document.
