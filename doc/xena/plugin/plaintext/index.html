About the Plain Text plugin

Xena can normalise plain text documents, which are unformatted text files often coded in ASCII, or .txt files created by Microsoft Notepad. When normalising plain text documents, Xena uses the plaintext plugin. This is included as part of the basic Xena application and does not require any third party software.

For how to use the plain text plugin to normalise a plain text document, see How to normalise plain text documents.

For how to use the plain text plugin to view a normalised plain text document, see How to view a normalised plain text document.
