About the dataset plugin

The dataset plugin is used to normalise any column and row type data into a structure we call datasets. File types that can be normalised into datasets include comma separated values (CSV) files and JDBC databases.

For how to use the dataset plugin to normalise a comma separated file (CSV file), see How to normalise a CSV File.

For how to use the dataset plugin to view a normalised Xena-Dataset, see How to view a normalised dataset.
