File Formats

Xena's ability to normalise documents is provided by the use of plugins. Each plugin enables Xena to normalise one or more data formats. Some plugins use third party software such as OpenOffice.org.

Below is a sample of the record types that Xena can currently normalise. For a definitive list, see the documentation on the individual plugins.

  • Microsoft Excel 5.0/95/97/2000
  • Microsoft Word 6.0/95/97/2000
  • Microsoft PowerPoint 97/2000
  • ASCII text documents with Unix, Windows, or MacOS line endings
  • Simple HTML documents
  • JPEG, GIF, TIFF and BMP images
  • CSV files
  • Email messages
  • Email boxes

The following formats have been tested and Xena cannot currently normalise them. When asked to normalise them it will treat them as binary objects and XML wrap the files. It is intended that future releases of Xena will be able to normalise these formats:

  • PDF documents
  • PCX images

As it is an open source application, any user of Xena may write a normalisation path for a new data format.
