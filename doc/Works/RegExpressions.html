Batch Filters

Regular Expression is the term Xena uses to describe its "batch filters". It is a language you can use to search the contents of a directory by file title or type using pattern matching. The search will select and return any files whose title or type matches the Regular Expression you create. This can be useful when the directory containing the files you want to normalise is large and you can clearly define the types of files you want to process.

Regular Expressions are constructed from a number of elements. These elements include the text you wish to match to and one or more constructs that denote a wildcard, whitespace or other element of the file name. See "Common Batch Filters" for a list of commonly used constructs.
