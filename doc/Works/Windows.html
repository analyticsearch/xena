Xena on Windows

Xena is a Java application and can run on any Windows operating system that can support a Java Runtime Environment. Xena has been thoroughly tested on Windows 2000.

Swapping between Xena and other applications when the batch processor window is open

There is an issue when using the batch processor on Windows OS that users should be aware of. As batch processing can take a considerable amount of time, it is common practise to switch to another application while batch processing runs in the background. When a user clicks on the Windows taskbar to return to Xena, the standard Xena interface is presented, not the batch processing window, and Xena appears to be frozen.

This is not a bug, it is normal behaviour for Java based applications. Xena has not frozen and the batch processor window is still open.

To return to Xena, use the ALT+TAB keys to switch tasks and select Xena. Do this by holding down the ALT key and repeatedly pressing the TAB key until Batch Processing is selected in the pop-up window.
