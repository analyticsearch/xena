Xena on Mac OSX

Xena is a Java application which runs on the Mac OSX operating system. Xena has been tested on Mac OSX, and all the major functions are operating. However, this testing has revealed a few issues with running Xena on Mac OSX.

Automatic file naming

The Save As dialog box does not correctly show the default output file name. This appears to be a bug in the Java Mac OSX look and feel.

If you encounter this problem, change the look and feel to Metal.

Set the look and feel to Metal

  1. From the Edit menu select the Preferences submenu.
  2. From the Preferences submenu select Look and Feel.
  3. From the Look and Feel submenu select Metal.

Known issues with MacOSX

Xena requires Java 1.4.2 to run on Mac OSX, which in turn runs on Mac OS10.3.4 or later.

OpenOffice.org on Mac OSX is only available for version 1.0, whereas Xena was designed to work with OpenOffice 1.1. It is not anticipated that Xena will support OpenOffice 1.0.
