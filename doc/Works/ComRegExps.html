Common Batch Filters

The following is a list of some Regular Expression elements that are useful within the Xena context. It is a small subset of the possible elements. For more information see http://java.sun.com/j2se/1.4.2/docs/api/java/util/regex/Pattern.html

Construct Matches
 
x The character x
\\ The backslash character
[abc] a, b, or c
[^abc] Any character except a, b, or c
[a-zA-Z] a through z or A through Z, inclusive
[a-d[m-p]] a through d, or m through p
[a-z&&[^bc]] a through z, except for b and c
[a-z&&[^m-p]] a through z, and not m through p: [a-lq-z]
. Any single character
\. A full stop
.* Any number of any characters - a wildcard
\d A digit: [0-9]
\D A non-digit: [^0-9]
\s A whitespace character
\w Any word character: [a-zA-Z_0-9]
\W Any non-word character, eg #$@^%
\b A word boundary
