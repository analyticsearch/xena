Xena on Linux

Xena is a Java application which runs on the Linux operating system. Xena has been tested on Linux RedHat 7.1 and RedHat 9, and all the major functions are operating.

There is no installation program for Linux and users must configure it manually.

Normalising PST files

Xena running on Linux cannot normalise PST files.
