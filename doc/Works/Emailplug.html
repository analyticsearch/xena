About the Email Plugin

Xena can normalise many email formats using the email plugin. This is included as part of the basic Xena application.

Email message formats that Xena can normalise include:

  • RFC822
  • MIME
  • Tower Software's TRIM email messages
  • MS Outlook's msg

It can also convert the following formats that form collections of messages:

  • MS Outlook pst data file
  • mbox data file

Restrictions on email normalisation

Xena can only normalise MS Outlook pst data file on Windows OS. This is because it calls an application (readpst.exe) that only works on Windows OS.
