Configuring Xena

Before using Xena you will need to do some basic configuration. Exactly what needs to be done will depend on your computing environment, but will always include setting the properties and base directories.

Setting the Xena properties

Before using Xena, you must set the properties. This includes choosing a package wrapper and file namer. See Setting the Xena Properties for instructions.

Setting the base directories in Xena

Before normalising a batch of records using Xena, you must set the base directories. See Setting the Base Directories for instructions.

Xena and operating Systems

As a Java application Xena is able to run on different operating systems. However, there are operating system specific issues with Xena and these are noted on the folliwng pages:

  • Windows
  • Mac OSX
  • Linux

Plugin specific configuration

Some plugins require plugin specific configuration or additional software to function properly. For example, the Office plugin requires that OpenOffice.org is installed on the computer and its installation directory is configured in Xena.

Each plugin has an 'About' page that details any specific configuration and software requirements for that plugin. You should read the 'About' page for each plugin before using it to normalise or perform a record.
