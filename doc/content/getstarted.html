Getting Started

Before using Xena 4.0, a couple of simple initial settings must be made. These are found via the Tools menu.

The location where files are to be stored after the conversion process must be specified. It is also useful to set a location for Xena's log file.

OpenOffice.org must be installed if Xena is to normalise such files as word processed documents, spreadsheets and presentations. Xena uses a 2.2.x version of OpenOffice.org, which may be downloaded freely from http://download.openoffice.org.

If OpenOffice.org 2.2.x is not installed then Xena will binary normalise these file types.

See the Set Preferences and Set Up Plugins sections below to configure these settings.
