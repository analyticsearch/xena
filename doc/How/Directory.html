Batch Normalise By Directory

When batch normalising a directory, Xena will guess the most appropriate file format and normalise the records using the relevant normaliser. When Xena does not have a normaliser that is specific to the particular file type within the directory, it will normalise it as a binary file.

To batch normalise by directory

  1. From the Xena File menu, select Batch Normalise.
  2. Click on the Source Directory Browse button.
  3. Navigate to and select the source directory. Click on the Open button.
  4. Click on the Destination Directory Browse button.
  5. Navigate to and select the destination directory. Click on the Open button.
  6. In the source pane select the directory you wish to normalise and click on » to copy the directory to the destination pane.
  7. Click on the Start button to commence batch normalisation.
