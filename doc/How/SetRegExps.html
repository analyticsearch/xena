Batch Filters

Batch Filters or Regular Expressions are used as part of the batch normalisation process to search the contents of a directory by file title or file type using pattern matching. The search will select and return any files whose title or type matches the Regular Expression you create. This expression can include wildcards.

It is important to note that Regular Expressions are case sensitive.
