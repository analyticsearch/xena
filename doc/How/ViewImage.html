How to view a normalised image

Xena uses the image plugin to view normalised Xena-PNG and Xena-JPEG files. The image plugin is part of the basic Xena application.

To view an image that has been normalised:

  1. Open Xena.
  2. From the File menu, select Open.
  3. Using the Open window, select the normalised image you wish to view and click on Open.
    Xena will display the image.

Multipage TIFF images are presented in the Xena multipage view. To move between the pages of the multipage Xena view, click on the various buttons. The First button will display the first page of the package. The Last button will display the final page of the package. The Previous and Next buttons will move between the previous and next pages. If you wish to go to a particular page, type the number in the Page box and press Enter.

View menu options

The View menu provides options for the current view of the current document. The options available under the View Menu will depend on the type of document that Xena is displaying, and the view of that document that you have chosen. The following sub-menu options are available for the Package view of images.

Zoom

This sub-menu allows you to enlarge or reduce the display size of the image to a certain percentage of the normalised image size. There are preset percentages, or you can enter a custom zoom percentage.

Fit

This sub-menu allows you to enlarge or reduce the display size of the image, based on the size of the active window. The options available are:

  • Fit to Size reduces or enlarges the image so that it fits neatly within the display window.
  • Fit to Width reduces or enlarges the image so that it fills the width of the display window.
  • Fit to Height reduces or enlarges the image so that it fills the height of the display window.
  • Fit to Both stretches or compresses the image so that it fits neatly within the display window, which can change the aspect ratio of the image.
