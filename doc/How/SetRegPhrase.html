Batch Filter By File Phrase

Often you will want to select a phrase rather than a single word. As an example, to normalise all files with the phrase "Parliament House" in the title the Regular Expression would be .*\bParliament\sHouse\b.* where:

  • .* means any number of any character, ie a wildcard for zero or more of any characters. We include it at both the beginning and the end so that we find the words "Parliament House" anywhere in the title.
  • \b marks a word boundary, ie the beginning or ending of a single word. We include it at the beginning and the end to denote the start and finish of the words we want.
  • "Parliament House" are the words we want to select for, so they included between the other elements.
  • \s denotes a whitespace, ie one space in the title. If you want to select files where the two words are separated by more than one space, use \b instead of \s

To select by phrase

  1. From the Xena File menu, select Batch Normalise
  2. In the Regular Expression box enter .*\b[word1]\s[word2]\b.*
  3. Click Go.

All files in the Source pane whose name contains the words entered in the Regular Expression box will be copied to the Destination pane.
