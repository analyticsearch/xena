Batch Filter By Word

A common type of searching is by a single word in the title. This could be used to select files relating to the same topic or containing a particular keyword in the file title. As an example, to normalise all files in a directory containing the word "minister" in the title the Regular Expression would be .*\bminister\b.* where:

  • .* means any number of any character, ie a wildcard for zero or more of any characters. We include it at both the beginning and the end so that we find the word "minister" anywhere in the title.
  • \b marks a word boundary, ie the beginning or ending of a single word. Without it we would include words such as "ministerial" or "administer". We include it at the beginning and the end to denote the start and finish of the word we want.
  • minister is the word we want to select for, so it is included between the other elements.

To select by word

  1. From the Xena File menu, select Batch Normalise.
  2. In the Regular Expression box enter .*\ b[word]\b.*
  3. Click Go.

All files in the Source pane whose name contains the word entered in the Regular Expression box will be copied to the Destination pane.
