How to view a normalised email message

Xena uses the email plugin to view normalised email documents. This plugin is part of the basic Xena application.

To view an email document that has been normalised:

  1. Open Xena
  2. From the File menu, select Open.
  3. Using the Open window, select the normalised email that you wish to view and click on Open.
    Xena will display the email document.

The normalised email display

The package view for an email is made up of three elements, the header, the body and any attachments. The header displays the usual email metadata elements - the From, To and Reply to addresses, the subject line, the date sent and Message ID. The body displays the textual content of the email. At the bottom of the display is a window for any attachments in the email. Click on the button to view the attachments.

View menu options

The View menu provides options for the current view of the current document. The options available under the View menu will depend on the type of document that Xena is displaying, and the view of that document that you have chosen. The view menu has several options when viewing normalised emails.

Wrap

This submenu allows you to choose how Xena wraps the displayed text. You may choose between having no text wrapping, wrapping at the character level, or wrapping at word level.

Set Tab Size

This sub-menu allows you to set the number of character spaces between each tab stop when displaying plain text documents.

Font

This sub-menu allows you to switch between monospace and unicode style fonts.
