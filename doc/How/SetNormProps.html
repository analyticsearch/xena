Set Normaliser Properties

This window allows you to choose which normalisers are to used and which are to be disabled. This is useful if you want Xena to guess the format of a document, but want to exclude certain normalisation options. For example you might not wish to binary wrap files if there is no other normaliser because you have already binary wrapped all of the files. By default all of the normalisers are active.

To change the normalisers properties

  1. From the Edit menu select Properties
  2. Click on the Normalisers tab
  3. The Normalisers properties are displayed

The left window displays all of the plugins that have been installed. The centre and right windows list the normalisers that plugin can use, in terms of data formats. The normalisers listed in centre window are the ones that are available and will be used and the normalisers listed in the right window are the ones that have been deactivated.

To disable a normaliser

  1. From the list of plugins select the plugin that contains the normaliser to be disabled.
  2. The normalisers available for that plugin will be displayed in the centre window, as shown:
  3. Select the normaliser you wish to disable and click on the >> button.
  4. Repeat these steps for each normaliser you wish to disable.
  5. Click OK to save your settings.

To reactivate a disabled normaliser

  1. From the list of plugins select the plugin that contains the normaliser you wish to reactivate.
  2. The deactivated normalisers available for that plugin will be displayed in the right window, as shown:
  3. Select the normaliser you wish to reactivate and click on the << button.
  4. Repeat these steps for each normaliser you wish to reactivate.
  5. Click OK to save your settings.
