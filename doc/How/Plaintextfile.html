Normalise Plain Text File

Xena can normalise plain text documents such as unformatted text files coded in ASCII, or .txt files created by Microsoft Notepad. When normalising plain text documents, Xena uses the plaintext plugin. This is part of the basic Xena application.

To normalise a plain text document

  1. Open Xena.
  2. From the File menu, select Normalise.
  3. Using the Open window, navigate to and select the document you wish to normalise.
  4. Click on Open.
    The Choose File Type dialog box will open, asking you to choose the file type of the document you are opening.
  5. Select PlainText from the drop down list and click OK.
    The configuration window will open, asking you to select the format of text that the document is coded in.
  6. From the drop down list, select the format of the text; if in doubt, accept the default offered by Xena.
  7. Click on the Finish button.
    The normalised plain text document will now be displayed by Xena.

The document has now been normalised into the Xena-Plaintext format.
