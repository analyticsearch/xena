Normalise PNG or JPEG File

Xena uses the image plugin to normalise PNG and JPEG images. This plugin is included as part of the basic Xena application.

To normalise a JPEG image

  1. Open Xena.
  2. From the File menu, select NormaliseFile.
  3. Using the Open window, navigate to and select the image you wish to normalise.
  4. Click on Open.
    The Choose File Type dialog box will open, asking you to choose the file type of the image file you are opening.
  5. Select PNG or JPEG from the drop down list, and click OK.
    The image will now be displayed by Xena.

The PNG or JPEG image has now been normalised into a Xena-PNG or Xena-JPEG file.
