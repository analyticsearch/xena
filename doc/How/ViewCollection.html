How to view a normalised collection of emails

Xena uses the email plugin to view normalised email documents and collections of emails. This plugin is part of the basic Xena application.

When it normalises a collection of emails, such as a .pst file, Xena noramlises each email file as an individual item. So to view an normalised collection of emails you will need to look at the individual emails. See How to view a normalised email message for how to do this.
