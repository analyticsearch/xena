Batch Normalise By File Type

You may wish to normalise files of one particular type, JPEG images for instance, within a directory that contains multiple file types. You can specify the normaliser to be used, the JPEG normaliser in this case, rather than relying on Xena's guessers. By using a Regular Expression you can sort the files by type prior to normalising them.

To batch normalise by file type

  1. From the Xena File menu, select Batch Normalise.
  2. Click on the Source Directory Browse button.
  3. Navigate to and select the source directory. Click on the Open button.
  4. Click on the Destination Directory Browse button.
  5. Navigate to and select the destination directory. Click on the Open button.
  6. In the Regular Expression box enter .*\.[file extension]
  7. Click the Regular Expression "Go" button.

    8. All files matching the type contained in the Regular Expression will now be copied from the Source pane to the Destination pane.

  8. Click the Start button to normalise all the selected files.
