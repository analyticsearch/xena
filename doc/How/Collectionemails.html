Normalise a Collection of Emails

Xena normalises collections of email messages (mailboxes) such as .pst files, using the email plugin. This plugin is part of the basic Xena application.

Xena can only normalise .pst files on Windows OS, because it calls an application (readpst.exe) to do this and that application only runs on Windows OS.

To normalise a collection of emails

  1. Open Xena.
  2. From the File menu, select Normalise File.
  3. Using the Open window, navigate to and select the .pst file that you wish to normalise.
  4. Click on Open.
    The Choose File Type dialog box will open, asking you to choose the file type of the document you are opening.
  5. Select Pst Outlook from the drop down list and click OK.
    The Mbox Dir Select Folder window will open.
  6. click on Finish.
    Xena will now normalise the email messages in the .pst file and displays a normalisation log.
    The email message will now be displayed by Xena.

The email message has now been normalised into the Xena email format.

The Mbox Dir Select Folder window has a Configuration menu, which gives you the option of loading a saved configuration or saving the current configuration. Similarly you can save the normalisation log if you wish.
