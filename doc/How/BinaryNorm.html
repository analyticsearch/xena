Binary Normalisation

This process converts each data file in a directory into Base64 and places an XML wrapper containing metadata around it.

To binary normalise a directory of files

  1. From the Xena File menu, select Batch Process.
  2. In the output directory select a group. You may only select one group at a time.
  3. Click on the Configure button.
  4. Under Normaliser Selection, click on Select.
  5. Select the Binary file type from the drop down list.
  6. Click OK.
  7. Repeat for each group selected.

Now that the batch process has been configured, click on the Start button to run the batch process.
