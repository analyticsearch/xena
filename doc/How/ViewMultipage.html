How to view a normalised multipage Xena package

Xena uses the multipage plugin to view normalised multipage Xena packages. This is part of the basic Xena application.

To view a multipage Xena package:

  1. Open Xena.
  2. From the File menu, select Open.
  3. Using the Open window select the normalised Xena-Multipage package you wish to view, and click on Open.

    Xena will display the multipage Xena package.

To move between the pages of the multipage Xena package, click on the various buttons. The First button will display the first page of the package. The Last button will display the final page of the package. The Previous and Next buttons will move between the previous and next pages. If you wish to go to a particular page, type the number in the Page box and press Enter.

View menu options

The View menu provides options for the current view of the current document. The options available under the View menu depend on the type of document that Xena is displaying and the view of that document that you have chosen.

The Multipage format itself has no custom View options, but assuming that you store images within your multipage, you will have the same View options available as you have for plain images. Other format types will have the relevant view option for their file type. For more information see the various help topics under Performing normalised files.
