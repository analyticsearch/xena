Normalise an Email

Xena normalises email messages using the email plugin. This plugin is part of the basic Xena application.

There are many different email formats. The text content of an email can be formatted as HTML, plain text or rich text. Xena normalises these all using the same procedure. One exception is .eml messages. These are normalised as plain text and you should use the plain text normalisation procedure for them.

Emails can also contain attachments. When Xena normalises an email with attachments they are stored as binary data.

To normalise an email message

  1. Open Xena.
  2. From the File menu, select Normalise.
  3. Using the Open window, navigate to and select the email message that you wish to normalise.
  4. Click on Open.
    The Choose File Type dialog box will open, asking you to choose the file type of the email you are opening.
  5. Select Microsoft MSG from the drop down list and click OK.
    The email message will now be displayed by Xena.

The email message has now been normalised into the Xena email format.
