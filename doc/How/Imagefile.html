Normalise Other Images

Xena uses an image plugin to normalise a range of image formats, including Tagged Image File Format (TIFF) and Graphic Interchange Format (GIF) into the Xena-PNG file format. The image plugin is included as part of the basic Xena application. If you open an image format that Xena cannot normalise it will XML wrap the image.

To normalise an image

  1. Open Xena.
  2. From the File menu, select Normalise File.
  3. Using the Open window, navigate to and select the image you wish to normalise.
  4. Click on Open.
    The Choose File Type dialog box will open, asking you to choose the file type of the image you are opening.
  5. Select Image from the drop down list and click OK.
  6. Click on Finish.
    The image will now be displayed.

The image has now been normalised as a Xena-PNG or Xena-JPEG file.
