How to view a normalised office document

Xena uses the Office plugin to view normalised Xena-Office documents. This is part of the basic Xena application. Xena cannot display office documents internally; it uses OpenOffice.org to display the documents. The Show in External Window button opens the document in OpenOffice.org.

To view a document that has been normalised:

  1. Open Xena.
  2. From the File menu, select Open.
  3. Using the Open window, navigate to and select the normalised document that you wish to view and click on Open.
    Xena will display the normalised office document
  4. At the top of the display window is a Show in OpenOffice.org button. Clicking on this button will open the document in OpenOffice.org

While the inbuilt viewer for normalised Office documents in Xena shows the textual content of document being displayed, it can lose some formatting information. Viewing the document in OpenOffice.org, will display this information and so it is the preferable view.

The document displayed in OpenOffice.org is a read-only document, and is completely independent of Xena. You can save it using OpenOffice.org, and make changes to this version. This will not affect the normalised version of the document. When you have finished viewing the document, close it both in OpenOffice.org and Xena.

View menu options

The View menu provides you with options for the current view of the current document. There are no selectable options in Xena for viewing Office documents.
