Batch Normalise By Word

Being able to select files by word allows you to choose to normalise all files in a large directory containing a particular keyword without having to select them manually. It also allows you to choose which plugin Xena will use when normalising a batch of files - particularly useful when all the files are of the same type.

To batch normalise files by word

  1. From the Xena File menu, select Batch Normalise.
  2. Click on the Source Directory Browse button.
  3. Navigate to and select the source directory. Click on the Open button.
  4. Click on the Destination Directory Browse button.
  5. Navigate to and select the destination directory. Click on the Open button.
  6. In the Regular Expression box enter .*b[word]\b.*
  7. Click the Regular Expression "Go" button.

    8. All files matching the word(s) contained in the Regular Expression will now be copied from the Source pane to the Destination pane.

  8. Click the Start button to normalise all the selected files.
