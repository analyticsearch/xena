Set Base Directories

The four base directories are all set using the same procedure. The procedure given below should be repeated for each directory you wish to set.

  1. From the Edit menu select Set Base Directories.
  2. The Set Base Directories window will be displayed:
  3. Click on the Browse button to the right of the directory you want to select.
  4. Navigate to the desired directory and click Open to accept it.
  5. Repeat this procedure for each of the base directories.
  6. Click OK when you have finished.

The directories that Xena will use for normalisation have now been selected.
At step 3 above you could just type in the file path of the desired directory if you wish, instead of using the browse button.

NOTE: The source and destination directories have no relationship to the default directories used by the Normalise and Open functions. These simply remember the last used directory.
