Normalise a TRIM Email

TRIM is an electronic document and records management system created by Tower Software. When emails are stored in the system they are converted into mbx format. Any attached files are also stored in the system. When the email is later exported from the system any attachments are exported with it. The Xena TRIM email normalisation process normalises the email together with any attachments that are part of it.

Xena normalises TRIM email messages using the email plugin. This plugin is part of the basic Xena application. When normalising TRIM emails you will need to point the TRIM normalisation processor at the folder containing the TRIM email. This way any attachments that are part of the email will also be captured into the email.

To normalise a TRIM email

  1. Open Xena.
  2. From the File menu, select Normalise File.
  3. Using the Open window, navigate to and select the folder containing the TRIM email message that you wish to normalise.
  4. Click on Open.
    The Choose File Type dialog box will open, asking you to choose a file type for the email that you are opening.
  5. Select TRIM email from the drop down list and click OK.
    The TRIM email message will now be displayed by Xena.

The TRIM email message has now been normalised into the Xena email format.
