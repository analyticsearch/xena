Batch Normaliser

While Xena can be used to normalise individual records, it is more likely you will want to automatically normalise all the records in a directory, especially when there may be thousands. Xena does this using batch normalisation. The Xena batch normalisation process goes through an entire directory, including sub-directories, and normalises all the records within that directory that it is able to normalise, according to the instructions that you have given it.

There are several ways of using the batch normaliser. You can select a single directory, and all the files within it, or you can use Regular Expressions - a batch filter to select all files with a particular file extension or file title from one or more directories.

You can rely on the Xena guessers to determine which normaliser to use, or you can specify the type of files you are normalising, and choose the normaliser to use. These decisions are made when you set up each batch normalisation run.

You use the Batch Processing Setup window to do batch processing in Xena. To open this window, from the File menu select Batch Process. This is what the batch processing setup window looks like:

» Show me

The batch processing window and the options it offers is explained in Batch Window Elements.

As Xena normalises files, the results will appear in the bottom window. When the batch normalisation is complete, the Stop button will grey out and a completion notice will be appended to the results list in the bottom window. The results window will show OK for each file that is properly normalised, ER if there is an error and NN for a warning message.
