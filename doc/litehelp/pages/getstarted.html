Getting Started

When Xena Lite is used for the first time, the location of the normaliser plugins directory and the location of OpenOffice.org must be specified.

The plugins directory is normally located in the same directory as the xena.jar file.

OpenOffice must be installed if Xena is to normalise office documents. Xena Lite uses a 1.1.x version of OpenOffice, which may be downloaded from OpenOffice.org.
