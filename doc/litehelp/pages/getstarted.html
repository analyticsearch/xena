Getting Started

Before using Xena Lite, a couple of simple initial settings must be made. These are found via Xena Lite's Tools menu.

OpenOffice must be installed if Xena is to normalise office documents. Xena Lite uses a 1.1.x version of OpenOffice, which may be downloaded from http://download.openoffice.org.

If OpenOffice 1.1.x is not installed, Xena will binary normalise office files.
