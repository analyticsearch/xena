Set up Directories

Xena Lite needs to know where normalised files should be saved after processing. Select Tools -> Xena 3.0 Lite Preferences from the main menu.

Use the Browse button to open a file browser and navigate to the location where normalised files should be saved.

The Preferences box may also be used to set a location for Xena Lite to save its log file. Xena Lite will continue to create a log if this preference is left blank, however specifying a location here may make locating the log file easier at a later date.
