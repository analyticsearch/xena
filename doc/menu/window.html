The Window menu

The window menu enables you to change between open windows and to open new windows. You can also change the type of view for an open normalised file.

Change View

This menu option enables you to change how Xena presents a record. When you first view a record, Xena uses the default view type for that record type. If your organisation wraps records with package metadata, the package will be the outermost view, containing a sub-view showing the record itself.

The default view presents the record in a manner that preserves as much as possible the original look and feel of the record. You can also view the record in other ways. Usually there will be a raw XML view of the record, in addition to the record type specific view or views. Because views can be embedded within views, Xena displays a dialog box asking which part of the view to change.

Changing Views

To change the view of a normalised record, open the record and then do the following:

  1. From the Window menu select Change View.
    The Change View window will open.
  2. Select the top item on the tree; for example Package View.
  3. Select the desired view from the drop down list at the bottom of the Change View window and click OK.
    The file will display in the desired view.

Clone

This menu option copies the active document window, opening a second version of the same document using the default view for that record type. This is useful if you wish to have two or more different views of the same file open at the same time.

Note: Clone will not clone the chosen view of the record

Close

This closes the active window.

Tile

This tiles all the active windows, displaying all of them at the same time.

View Normaliser Log

This menu option allows you to view the normalisation log file for a recently normalised file. If you close a file after normalising it you will not be able to view the normalisation log file, unless you save it. To save an open log file, from the File menu select Save Log.

List of open documents

At the bottom of the Window menu, Xena lists all of the open documents. The active document has a bullet point next to it on the menu. Select the chosen window to activate it.
