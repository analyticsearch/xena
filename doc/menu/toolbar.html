The Xena toolbar

The Xena toolbar provides quick access to the most commonly used functions of Xena.

Normalise

This toolbar button enables you to normalise any file type supported by Xena. See Normalise File for more information on this option.

Export

This toolbar button allows you to export a Xena file in its native format. For instance, an Office document that has been normalised would be exported as an OpenOffice.org document, or a normalised image would be exported as a PNG image. See How to Export Xena Files for how to use this toolbar button.

Normalise Non-File

Xena usually normalises data objects by accessing them as data files through a filesystem. This toolbar button allows you to access data objects through other protocols such as JDBC (for relational database systems) and HTTP (for web resources).

Batch Process

This toolbar button is used to open the batch normalisation process. The batch normalisation process is the most common way to normalise records, as it handles entire directories. For details on how to normalise a batch of records, see Normalising Batches of Records.

Open

This toolbar button enables you to open a Xena file for viewing. See Open for more information on this option

Save As

This toolbar button is used to save a document once it has been opened in Xena. It is used as part of the normalisation process to save non-normalised documents that have been opened in Xena. It will always save a document as a Xena XML package, ie a normalised record. For details on how to normalise a record, see the appropriate section for the type of record you want to view or normalise.

Set Base Directories

This toolbar button enables you to set the base directories in Xena. For more information on the base directories and how to set them see Xena and Base Directories.

Clone

This toolbar button copies the active document window, opening a second version of the same document using the default view for that record type. Note: Clone will not clone the chosen view of the record

Tile

This tiles all the active windows, displaying all of them at the same time.

View Log

This toolbar button allows you to view the normalisation log file for a recently normalised file. If you close a file after normalising it you will not be able to view the normalisation log file, unless you save it. To save an open log file, from the File menu select Save Log.

Change View

This toolbar button enables you to change how Xena presents a record. See Change View for more information on this option.

Help Topics

This toolbar button displays the user documentation, which includes information on how to use Xena as well as Xena-supported format specifications.
