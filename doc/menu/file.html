The File menu

Normalise File

This enables you to normalise any file type supported by Xena.

When you open a file for normalising, Xena uses its guessers to guess its file type. It does this by sequentially checking if the file conforms to the known properties of a particular file type (file extensions, file layout, etc). It uses all of its guessers in sequence, then suggests the possible options, with its best guess as the default option. If the file is a Xena file, the guess should always be correctly identified. Other types of files will vary in the guessing accuracy. Once the file type is known, Xena will choose a normaliser that can normalise that file type. If the file type is already Xena, Normalise Files just opens it for viewing.

For details on how to normalise a file, see the appropriate section for the plugin for the type of file you want to normalise.

Export

This option allows you to export a Xena file in its native format. For instance, an Office document that has been normalised would be exported as an OpenOffice.org document, or a normalised image would be exported as a PNG image.

See How to Export Xena Files for how to use this menu option.

Normalise Non-File

Xena usually normalises data objects by accessing them as data files through a filesystem. This option allows you to access data objects through other protocols such as HTTP (for web resources).

Batch Process

This menu option is used to open the batch normalisation process. The batch normalisation process is the most common way to normalise records, as it handles entire directories.

For details on how to normalise a batch of records, see Normalising Batches of Records.

Open

This enables you to open a Xena file for viewing.

Save As

This menu option is used to save a copy of a normalised file. While the normalisation process automatically saves the normalised file into the default location with a name determined by the file namer, you may wish to save it in a different location and give it a different name. It will always save a file as a Xena XML package, ie a normalised record.

For details on how to normalise a record, see the appropriate section for the type of document you want to view or normalise.

Exit

This menu option closes Xena.
