How to Export Xena Files

The Export menu option allows you to export a Xena file in its native format. For instance, an HTML document that has been normalised would be exported as an HTML document, or a normalised image would be exported as a PNG image.

xena's ability to export does vary with different data formats. It cannot export emails or datasets. Its ability to export other data formats are as follows:

Office documents such as word, powerpoint and excel files are exported as OpenOffice flat XML.

HTML documents are exported as html, however you will need to add an html extension to the file name.

Plain text documents are exported as plain text,Xena will include the file extension.

Image files such as bitmaps, JPEGs and GIFs are exported as PNG images.

binary normalised files are exported, however you will need to add the correct extension to the file name to beable to read it. ie, if it was a pdf file, adding .pdf to the end of the file will enable you to open and read it.

How to export Xena normalised files

Use the following procedure to export files from Xena

  1. from the file menu select export
  2. navigate to and select the file to be exported
  3. click on open
  4. the Save window will open
  5. navigate to and select the location where you want to save the file to be exported
  6. click on save to save the file

If you wish to change the file's name then you can do so before you save it. Just be sure not t change the file extension otherwise you may render the file unreadable.

You should now be able to open up the file just as you would with any other file on your computer. You will of course need an application that can read it.
