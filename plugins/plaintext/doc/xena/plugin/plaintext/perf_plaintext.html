How to view a normalised plain text document

Xena uses the plain text plugin to view normalised plain text documents. This is part of the basic Xena application.

To view a document that has been normalised:

  1. Open Xena.
  2. From the File menu, select Open.
  3. Using the Open window, select the normalised document you wish to view, and click on Open.
    Xena will display the plain text document.

It is possible to view the document as Raw XML or an XML Tree. See Change View under Xena Menus - The Window Menu for details on how to do this.

View menu options

The View menu provides options for the current view of the current document. The options available under the View Menu will depend on the type of document that Xena is displaying, and the view of that document that you have chosen. The following sub-menu options are available for the Package view of plain text documents.

Wrap

This submenu allows you to choose how Xena wraps the displayed text. You may choose between having no text wrapping, wrapping at the character level, or wrapping at word level.

Set Tab Size

This sub-menu allows you to set the number of character spaces between each tab stop when displaying plain text documents.

Font

This sub-menu allows you to switch between monospace and unicode style fonts.
