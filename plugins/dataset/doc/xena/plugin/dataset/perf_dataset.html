dataset plugin - viewing normalised datasets

How to view a normalised dataset

Xena uses the dataset plugin to view normalised datasets. This is part of the basic Xena application.

To view an dataset that has been normalised:

  1. Open Xena.
  2. From the File menu, select Open.
  3. Using the Open window, select the normalised dataset that you wish to view, and click on Open.
    Xena will display the dataset using the default table view.

The dataset package view has several sub views. The default is the Table view. You can also select Form and Properties views. To change between the various sub-views:

  1. From the Window menu, select Change View.
    The Change View window will open.
  2. Select the current view under Package View.
    In the example above that is the Table View.
  3. Select the desired sub-view from the drop down list at the bottom of the Change View window, and click OK.

The Table view displays the data as a data table. You can rearrange the data in the display by clicking on the column headings. This will sort the data by data order in that column.

The Form view displays the data as a series of pages, one page for each entry in the dataset, in the same interface as a multipage document. To view the individual entries, move between the pages by clicking on the buttons. The First button displays the first page of the package. The Last button moves to the final page of the package. The Previous and Next buttons move to the previous and next pages. If you wish to go to a particular page, type the number in the Page box and press Enter.

The Properties view displays the properties of the different columns in the dataset.

You can also view the dataset as Raw XML or an XML Tree. See Change View under Xena Menus - The Window Menu for details on how to do this.

View menu options

The View menu provides options for the current view of the current document. The options available under the View menu depend on the type of document that Xena is displaying, and the view of that document that you have chosen. The view menu has no options when viewing normalised datasets.
