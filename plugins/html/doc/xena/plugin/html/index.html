About the HTML plugin

Xena can normalise documents that meet the W3C definition for HTML. Using the HTML plugin. This is included as part of the basic Xena application, and does not require any third party software.

For how to use the HTML plugin to normalise an HTML document, see How to normalise HTML documents.
For how to use the HTML plugin to normalise a HTTP web site, see How to normalise a HTTP web site.

For how to use the HTML plugin to view a normalised Xena-HTML document, see How to view a normalised HTML document.

The HTML normaliser only normalises single, simple HTML pages. It does not normalise or include any associated images or other resources attached to a HTML page. However when you normalise a HTTP web site you can select to normlaise the resources associated with it, such as images.
