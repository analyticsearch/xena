How to view a normalised HTML document

Xena uses the HTML plugin to view normalised HTML documents. This plugin is part of the basic Xena application.

To view an HTML document that has been normalised:

  1. Open Xena
  2. From the File menu, select Open.
  3. Using the Open window, select the normalised document that you wish to view and click on Open.
    Xena will display the HTML document.
  4. At the top of the display window is a Show in Browser Window button. Clicking on this button will open the document in your default HTML browser

NOTE: Any images will not be displayed because the HTML normaliser only normalises the HTML file itself, not the images.

While the inbuilt viewer for normalised HTML documents in Xena shows the textual content of document being displayed, it can lose some formatting information. Viewing the document in an HTML browser will display this information and so is a preferable view.

You can also view the document as Raw XML or as an XML Tree. See Change View under Xena Menus - The Window Menu for details on how to do this.

View menu options

The View menu provides options for the current view of the current document. The options available under the View menu will depend on the type of document that Xena is displaying, and the view of that document that you have chosen. The view menu has no options when viewing normalised HTML documents.
