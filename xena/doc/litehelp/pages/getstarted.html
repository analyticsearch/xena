Getting Started

Before using Xena 3.0 Lite, a couple of simple initial settings must be made. These are found via Xena Lite's Tools menu.

The location where files are to be stored after the conversion process must be specified. It is also useful to set up a location for Xena's log file.

OpenOffice.org must be installed if Xena is to normalise files such as word processed documents, spreadsheets and presentations. Xena Lite uses a 2.0.x version of OpenOffice.org, which may be downloaded from http://download.openoffice.org.

If OpenOffice.org is not installed, Xena will binary normalise these files.
